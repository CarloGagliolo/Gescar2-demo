# Gescar2 — Demo

Demo interattiva statica del gestionale **Gescar2** per centri revisioni auto.

> 🔗 **[Apri la demo](https://carlogagliolo.github.io/Gescar2-demo/)**
>
> Tutti i dati mostrati sono completamente fittizi.

---

## Cosa mostra questa demo

- **Dashboard** — KPI scadenze imminenti con badge semaforo (rosso/giallo/blu/verde)
- **Scadenze** — lista filtrata per periodo con tab per tipo (revisioni, assicurazioni, bolli, patenti)
- **Clienti** — anagrafica con ricerca live per nome o targa
- **Scheda cliente** — veicoli intestati con stato scadenze e storico notifiche
- **Registro notifiche** — log avvisi con canale (telefono, SMS, email, WhatsApp) ed esito

---

## Stack

PHP 8.2 · Symfony 7.3 · Doctrine ORM 3 · MySQL 8.0 · Bootstrap 5.3

---

## Avvio locale

```bash
git clone https://github.com/CarloGagliolo/Gescar2-demo
cd gescar2-demo
open index.html   # oppure trascina il file nel browser
```

Nessuna installazione, nessun server, nessuna dipendenza. È un file HTML statico.
